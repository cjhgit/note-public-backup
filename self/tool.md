# 不要陷入形式的误区

一个好的工具或方法往往能起到事半功倍的作用，但有一个现象，很多入门者，包括我，容易陷入一个误区。

比如，很多入门者把大部分时间浪费在寻找工具，试用，再寻找，再试用的恶性循环中。我想开始记笔记，于是百度“怎么做笔记”，看到有人推荐了为知笔记，便下载使用，但总感觉这个软件不太好用。看到有人推荐 One Note，又去折腾 One Note。最后把主流的比较软件都试了一遍，笔记却没有记多少。

再如，我想写日记，我会搜索“怎样写日记”，但这次不是浪费时间在软件上，而是花了一堆时间在乱七八糟的东西上面：什么手账啊，日记模板啊，晨间日记啊，最后也没坚持写几天日记。

仔细想想，个人而言，其实还有很多这种思维误区：

* 纠结那种编程语言更好，却没有一门擅长的语言。
* 买了数位板，就坚持了两个月的画画。
* 看了健身的各种方法，坚持了半个月就坚持不下去了。
* 书没看多少，在想要不要买 kindle。
* ...

在思考这些的时候，我想到了一个词：“形式大于内容”。

我搜索了这个词，没找到什么资料。但我觉得这个词很好地概括了上面的情况。

做笔记时，笔记是内容，软件是形式；写日记时，日记本身是内容，方法是形式；读书时，读到的知识是内容，kindle 是形式。

当然，并不是说形式不重要，相反，方法和工具都是提高效率的至关重要的东西。只是，如果花在形式上的时间多于花在内容上，那么就有问题了。

前一段时间，有朋友说想买 kindle，我建议他，先在手机上阅读，如果能坚持两个月再去买。因为我知道，他没有坚持阅读的习惯，甚至读过书书都没几本，买来的 kindle 最后肯定是放在角落里吃灰。事实上也确是如此，他没能坚持下来，想买 kindle 也是当初的五分钟热度罢了。

先放弃形式，专注内容，如果能坚持下来，再去考虑形式。我觉得这是一个很好的方法。

不用急着买健身卡，先坚持基本的室内运动两个月，如果能坚持下来，再去买也不迟；如果你想记笔记、写博客，或是写写其他文字，一个电脑自带的记事本或者一支笔和一个普通本子就足够，其他的，坚持两个月后再说吧。

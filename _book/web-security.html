网络安全

  • DDoS（Distributed Denial of Service，分布式拒绝服务）攻击
  • DOS（Denial of Service，拒绝服务）攻击
  • XSS 和 CSRF

DNS 劫持：DNS 服务器拦截域名解析的请求，返回错误的 IP（一般是广告页面、钓鱼网站的 IP）或者不做任何响应。

HTTP 劫持：HTTP 请求时，在返回的内容中插入恶意代码。在网站发给你信息前就给你返回了请求(?)。

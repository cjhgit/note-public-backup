# Windows

## 端口占用查看

查找看当前系统的端口使用情况：

```
netstat -ano
```

可在【任务管理器】->【服务】杀死进程，或右键，选择【打开服务】，可以设置/关闭服务器自启动。


服务名：

* Apache2.4
* MySQL57
* Redis
